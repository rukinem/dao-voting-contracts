# 🏛️ dao-voting-contracts

Gas-optimized voting contracts with snapshot, quadratic and conviction voting

## Features

- Snapshot voting
- Quadratic voting
- Conviction voting
- Gas optimized
- Delegation

## Tech Stack

- **Smart Contracts:** Solidity ^0.8.20
- **Framework:** Hardhat
- **Frontend:** Next.js + TypeScript
- **Web3:** ethers.js / wagmi

## Getting Started

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network localhost
```
